Source code

Eina is open source. You can take our code and compile by yourself.

Get current version Eina-${eina_version_fix}
post 0.9.0 releases on launchpad
pre-0.9.0 releases on sourceforge

Ubuntu

Thanks to Canonical and Launchpad we can offer packages for the current version of Ubuntu (oneiric, 11.10). The package is named 'eina-player'

Add those line to your /etc/apt/sources.list or use Software sources to do it.

deb http://ppa.launchpad.net/eina/ubuntu oneiric main
deb-src http://ppa.launchpad.net/eina/ubuntu oneiric main

Or just run this command on a terminal: 

sudo apt-add-repository ppa:eina/ppa

Packages for older versions of Ubuntun can be found at Launchpad

Debian

Probabily our Ubuntu packages are (not 100%) compatibles with Debian Sid, we cannot warrant anything else.

Fedora

Eina included some versions of Eina since F9, but it's orphaned now. You can get the packages from its reposity

Arch Linux

`Official' Eina packages are available in section unsupported::multimedia

ZenWalk Linux

`Official' packages are available for Zenwalk or it can be installed with the following command:

netpkg eina-audio-player

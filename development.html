Git repository

Our repository is hosted at gihub.com:

git clone git://github.com/ldotlopez/eina.git

Currently Eina development is done on the `master' branch which uses gtk+-3 and bleding edge packages.
If you want to do some hacks with lower requeriments you can use some of the 0.9.x braches, they use gtk+-2. Those branches DON'T get updates.

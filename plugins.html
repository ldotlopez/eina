Muine

Are you missing a library mode? No problem, the "muine"(*) plugin provides a (reasonably good) library interface for Eina.

It stores the songs you listen and groups them by artist or by album, allowing you to search play or queue them.

* The name of this plugin is a tribute to an ancient music player called "Muine"
